Detailed Description

SHT31 represents an ADAfruit (or compatible) SHT31 temperature & humidity sensor.

The constructor creates a pointer to the sensor, and starts up the sensor. The pointer is passed to SHT31value, which retrieves the specified value.

Parameters
addrThe memory address of the sensor. Defaults to 0x44, but other addresses are available for some SHT31 chips. See the datasheet.
