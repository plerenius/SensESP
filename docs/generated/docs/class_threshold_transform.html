Detailed Description

template<typename C, typename P>
class ThresholdTransform< C, P >

A Transform base class that translates the value of type C into value of type P. Base class for classes NumericThreshold and IntegerThreshold.

Parameters
min_valueMinimum value of input for output to be the value of in_range.
max_valueMaximum value of input for output to be the value of in_range.
in_rangeOutput value if input value is in range.
out_rangeOutput value if input value is out of the range.
